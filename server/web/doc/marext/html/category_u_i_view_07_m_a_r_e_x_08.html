◆ mar_convertPoint:fromViewOrWindow:() [1/2]

- (CGPoint) mar_convertPoint: (CGPoint)  point
fromViewOrWindow: (nullable UIView *)  view 

Converts a point from the coordinate system of a given view or window to that of the receiver.

参数
pointA point specified in the local coordinate system (bounds) of view.
viewThe view or window with point in its coordinate system. If view is nil, this method instead converts from window base coordinates.
返回
The point converted to the local coordinate system (bounds) of the receiver.
