◆ mar_getNetworkTrafficBytes:()

- (uint64_t) mar_getNetworkTrafficBytes: (MARNetworkTrafficType)  types

Get device network traffic bytes.

This is a counter since the device's last boot time. Usage:

uint64_t bytes = [[UIDevice currentDevice] getNetworkTrafficBytes:MARNetworkTrafficTypeALL]; NSTimeInterval time = CACurrentMediaTime();

uint64_t bytesPerSecond = (bytes - _lastBytes) / (time - _lastTime);

_lastBytes = bytes; _lastTime = time;

参数
typestraffic types
返回
bytes counter.

(MARNetworkTrafficType)   

Network traffic type:

WWAN: Wireless Wide Area Network. For example: 3G/4G.

WIFI: Wi-Fi.

AWDL: Apple Wireless Direct Link (peer-to-peer connection). For exmaple: AirDrop, AirPlay, GameKit.
