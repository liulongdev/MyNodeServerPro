Abstract creation of a block-backed UITapGestureRecognizer.

This method allows for the recognition of any arbitrary number of fingers tapping any number of times on a view. An instance of UITapGesture recognizer is allocated for the block and added to the recieving view.

警告
This method has an additive effect. Do not call it multiple times to set-up or tear-down. The view will discard the gesture recognizer on release.
参数
numberOfTouchesThe number of fingers tapping that will trigger the block.
numberOfTapsThe number of taps required to trigger the block.
blockThe handler for the UITapGestureRecognizer
